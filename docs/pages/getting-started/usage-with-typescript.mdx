# Usage with Typescript

`@nfid/identitykit` uses exports keyword in package.json to make usage of imports convenient, but it
has some requirements to use:

- `tsconfig.json` module set to `node16` or `nodenext` or any other with moduleResolution set to
  `node16` or `nodenext`
- Typescript version >= 4.7
- Node.js version >= 12.7.0

```typescript
{
    "compilerOptions": {
        "module": "node16",
        // or
        "moduleResolution": "node16"
    }
}
```
