# Featured vs Compact mode

By default, IdentityKit's `modalMode` is set to `featured` (and looks like this):

![NFID Featured Mode](../../../docs/public/img/featured-mode-light.png)

```typescript
import { IdentityKitProvider } from "@nfid/identitykit/react"
import { NFIDW } from "@nfid/identitykit"

export const App = () => (
  <IdentityKitProvider
    featuredSigner={[NFIDW]}
    {...etc}
  >
    {/* Your App */}
  </IdentityKitProvider>
);
```

Available options for `featuredWallet`:

- `NFIDW`

We also provide a compact mode, which looks like this:

![NFID Compact Mode](../../../docs/public/img/compact-mode-light.png)

To set your modal size to `compact`, just leave featuredSigner empty. IdentityKitProvider:

```typescript
import { IdentityKitProvider } from "@nfid/identitykit/react"

export const App = () => (
  <IdentityKitProvider
    {...etc}
  >
    {/* Your App */}
  </IdentityKitProvider>
);
```
