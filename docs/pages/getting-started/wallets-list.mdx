# Customizing the list of wallets you support

By default, IdentityKit's `walletsList` is set to the order in which wallets adopted ICP wallet
standards:

- NFID Wallet
- Plug Wallet
- Oisy Wallet
- Internet Identity

To reorder your wallets list, just add `walletsList=[walletsList]` as a prop in your
IdentityKitProvider:

```typescript
import { IdentityKitProvider } from '@nfid/identitykit';

export const App = () => (
  <IdentityKitProvider walletsList=[walletsList] {...etc}>
    {/* Your App */}
  </IdentityKitProvider>
);
```
