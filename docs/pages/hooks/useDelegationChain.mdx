# useDelegationChain

Hook for getting the delegation chain for the currently connected user, returns:

- `getDelegationChain` - function to trigger get delegation from storage
- `delegationChain` - `undefined{:js}` for not connected user or `DelegationChain` for connected
  user
- `delegationChainError` - will include error if `getDelegationChain` was called for not connected
  user or if selected `IdentityKitAuthType.ACCOUNTS` auth type

## Import

```typescript
import { useDelegationChain, useIdentityKit } from "@nfid/identitykit/react"
```

## Usage

```jsx
const { delegationChain, delegationChainError, getDelegationChain } = useDelegationChain()
const { connectedAccount } = useIdentityKit()

useEffect(() => {
  if (connectedAccount) getDelegationChain()
}, [connectedAccount, getDelegationChain])
```
